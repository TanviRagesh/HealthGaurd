Project Overview

HealthGuard is an AI-powered early disease detection system designed to identify silent and undiagnosed health risks before they become life-threatening.

The platform collects user health data such as symptoms, lifestyle habits, family medical history, and daily health progress. Using intelligent risk analysis, HealthGuard predicts potential diseases and alerts users early, enabling preventive healthcare instead of reactive treatment.
